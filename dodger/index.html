운석 피하기

좌우로 이동해서 떨어지는 운석을 피하세요. 모바일은 아래 방향 버튼을 길게 눌러 조작할 수 있습니다.

점수0
생존 시간0.0s
상태대기
시작을 누르고 방향키(← →) 또는 A/D로 이동하세요.

랭킹

    게임 방법

    1. 시작 버튼을 탭하면 운석이 위에서 떨어지기 시작합니다.
    2. 좌·우 버튼(또는 키보드 방향키)으로 캐릭터를 이동해 운석을 피합니다.
    3. 운석에 닿으면 즉시 종료되며 생존 시간이 기록됩니다.
    4. 시간이 지날수록 운석 밀도가 높아집니다.
