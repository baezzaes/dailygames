DailyGames

모바일에서 바로 즐길 수 있는 미니게임 허브. 게임을 고르면 각 폴더의 개별 페이지로 이동합니다.

게임 선택

탭해서 시작
TOUCH 10초 클릭 챌린지 짧고 강한 탭 게임. 10초 동안 최대 클릭 수 기록. TIMING 반응속도 테스트 5라운드 평균 반응속도 측정. 모바일 탭 플레이 가능. GRAPHIC 운석 피하기 Canvas 그래픽 게임. 좌우 이동으로 운석 회피. MEMORY 색상 기억 게임 패턴을 기억하고 같은 순서로 탭하는 기억력 게임. RUNNER 피하기 + 코인 먹기 장애물을 피하고 코인을 먹어 점수를 올리는 그래픽 게임. TIMING 정지 타이밍 게임 움직이는 바를 목표 구간에서 멈춰 연속 성공 기록. SPEED 숫자 순서 탭 1부터 25까지 순서대로 탭, 짧은 기록이 유리. RHYTHM 레인 탭 내려오는 노트를 히트 라인에서 맞추는 4레인 탭 게임.
