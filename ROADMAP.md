# DailyGames 업그레이드 로드맵

출퇴근/점심 짬 시간에 10초~1분 안에 즐기는 모바일 미니게임 허브를 목표로 개선 중.

---

## 완료된 작업
- 최초 진입 닉네임 전용 화면 도입
- 닉네임 1회 입력 후 허브/게임 전역 자동 적용 (`window.prompt` 제거)
- 허브 + 게임 페이지 전면 모바일 리프레시
- 운석 피하기 모바일 화면 크기 개선
- 오늘의 도전(Daily Challenge) 도입
- 허브에서 게임별 개인 최고 기록(PB) 표시
- 게임 종료 직후 빠른 재시작 버튼 노출
- 허브 카드에 랭킹 TOP3 티저 표시
- 게임 종료 시 `오늘 N위` 즉시 피드백
- 결과 공유/카드 공유 기능
- PWA(홈 화면 추가) 동작 정리
- PB 초기화 목록 누락 버그 수정 (shadow, balance)
- 관리자 인증 헤더 방식 일원화 (query string 제거)
- 점수 저장 시 닉네임 태그 누락 버그 수정
- 연속 플레이(Streak) 표시 및 마일스톤(3/7/14/30일) 반영
- 게임 카드 플레이 시간 배지 추가 (`⚡ 10초 / ⏱ 30초 / 🎯 1분`)
- 허브 상단 `오늘 랭킹 마감` KST 카운트다운 추가
- 시간대 인식 허브 서브타이틀 추가 (출근/점심/퇴근)
- 닉네임 비속어/성적 표현 필터링 추가 (허브/게임 공통)
- 주간 랭킹 도입
  - API: `/api/rank` `mode=weekly` 지원
  - 기준: KST 월~일, 이번 주 실시간 집계
  - 규칙: 닉네임당 개인 최고 1개 반영, 동점 시 달성 시각 오름차순
  - 노출: 허브 + 10개 게임 화면 `오늘/주간` 토글 (기본 `오늘`)
- 캐시 대응: 허브/게임 페이지 CSS 로드에 버전 쿼리(`?v=...`) 적용
- 캐시 대응: 10개 게임 페이지 공통 `game.js` 로드에도 버전 쿼리(`?v=...`) 적용
- QA 체크리스트 문서 추가: `QA_CHECKLIST.md`

## 백로그 (다음 우선순위)
- [ ] 게임 종료 후 다음 게임 추천
- [ ] 종합(전체/시즌) 랭킹 도입
- [ ] Web Push 알림 (`오늘 랭킹 마감 n시간 전`)
- [ ] 관리자 페이지 주간 랭킹 탭 확장
- [ ] 정적 자산 캐시 정책 고도화 (해시 파일명 또는 빌드 기반 버저닝)

---

## 운영 메모
- 핵심 사용자 맥락: 출퇴근/점심의 짧은 플레이 시간(약 5~60초)
- 핵심 경험 축: 매일 리셋되는 일간 경쟁 + 이번 주 누적 체감(주간 랭킹)
- 품질 기준: 모바일 first, 빠른 재도전, 랭킹/닉네임 정책 일관성
