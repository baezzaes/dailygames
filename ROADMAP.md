# DailyGames 업그레이드 로드맵

출퇴근 시간 캐쥬얼 랭킹 경쟁 게임 사이트 개선 아이디어

---

## ✅ 완료
- 최초 방문 시 닉네임 입력 전용 첫 화면
- 닉네임 저장 및 게임 전반 자동 적용 (window.prompt 제거)
- 허브 + 게임 페이지 모던 디자인 리프레시
- 운석 피하기 모바일 화면 크기 개선 (세로형 비율 + 내부 해상도 동기화)
- **④ 오늘의 도전 (Daily Challenge)** - 매일 특정 게임 + 목표 점수 제시, 달성 시 시각적 보상
- **⑥ 내 개인 최고 기록 (PB) 표시** - 허브에서 각 게임별 내 기록 표시
- **⑧ 빠른 재시작** - 게임 종료 즉시 상단에 큰 "다시 하기" 버튼 노출
- **① 허브 랭킹 티저** - 각 게임 카드에 오늘 1위 점수 표시
- **② 내 순위 즉시 표시** - 게임 종료 후 "현재 3위!" 피드백
- **③ 점수 공유 버튼** - 카카오 / 링크 복사로 SNS 공유
- **⑪ PWA** - 홈 화면 추가, 앱처럼 설치 가능
- PB 초기화 목록 누락 버그 수정 (shadow, balance)
- 관리자 인증 헤더 방식 일원화 (query string 제거)
- 점수 저장 시 닉네임+태그 절삭 버그 수정
- **⑤ 연속 플레이 스트릭 (Streak)** - N일 연속 플레이 🔥, 마일스톤 뱃지(3/7/14/30일), 끊기기 직전 강조

---

## 📋 백로그

### 임팩트 높음 / 난이도 중간
- [ ] **⑦ 게임 완료 후 다음 게임 추천** - 이탈 방지, 세션 연장
- [ ] **⑨ 게임 소요 시간 표시** - "한 판 약 30초" - 바쁜 유저도 도전 유도
- [ ] **⑩ 종합 랭킹** - 모든 게임 점수 정규화 합산, "오늘의 챔피언" 선정
- [ ] **⑫ Web Push 알림** - "오늘 랭킹 마감 2시간 전!" 재방문 유도

---

## 💡 메모
- 타깃: 출퇴근 중 짧은 시간(5~60초) 캐쥬얼 플레이
- 핵심 훅: 매일 바뀌는 랭킹 + 경쟁 → 재방문 유도
- 오프라인/느린 네트워크 대응 중요 (지하철)
